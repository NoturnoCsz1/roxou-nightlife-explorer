# modules/motorista

**Domínio:** Painel do motorista (novo — não existe ainda).

Responsabilidades futuras deste módulo:

- Dashboard do motorista.
- Corridas aceitas / em andamento / concluídas.
- Ganhos, fechamento de turno, financeiro.
- Custos (combustível, energia elétrica, manutenção).
- Veículo(s) do motorista.
- Jornada de trabalho.
- Metas, demanda regional, relatórios, insights.
- Assinaturas / plano do motorista.

Não pertence a este módulo: aceite operacional de corrida em tempo
real (isso fica em `modules/transporte/caronas`), nem gestão de
excursões (fica em `modules/transporte/excursoes`).

**Status:** pasta criada. Nenhum código escrito ainda — esse módulo
será construído do zero em uma onda dedicada, reaproveitando o
`pricing` extraído de `V3RideRequest`.
